Ежеденевник
Проект, в котором пользователь входит свою учетеную запись
В своем аккаунту заполняет список дел на каждый день недели.
В конце дня есть возможность отметить выполение заплонированных дел. На основе этого будет выводиться статистика в процентах(на сколько пользователю удается придерживаться своего плана на день).
